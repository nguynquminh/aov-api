Giới thiệu

API này cung cấp dữ liệu về các yếu tố trong game Liên Quân Mobile bao gồm tướng, trang bị, ngọc, phép bổ trợ và huy hiệu. Tất cả các endpoint đều trả về dữ liệu dưới dạng JSON.

Kiểm tra API Click để kiểm tra kết nối tới server

 Ví dụ sử dụng

                
Dưới đây là một số ví dụ về cách sử dụng API này:

                
                

                    
JavaScript (Fetch API)

                    
// Lấy danh sách tướng
fetch('/api/heroes')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

// Tìm kiếm trang bị
fetch('/api/equipments/search?q=kiếm')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

                

                
                

                    
Python (Requests)

                    
import requests

# Lấy danh sách ngọc
response = requests.get('http://your-domain.com/api/runes')
data = response.json()
print(data)

# Tìm kiếm phép bổ trợ
search_response = requests.get('http://your-domain.com/api/spells/search?q=trừng')
search_data = search_response.json()
print(search_data)
